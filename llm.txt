# btwfyi - LLM Integration Guide

> Task awareness overlay for development environments.

## Quick Start

```bash
pnpm add @remcostoeten/btwfyi
```

```tsx
import { Btwfyi } from '@remcostoeten/btwfyi/react'

<Btwfyi category="dev" categories={categories} />
```

## Smart Syntax

Parse natural language: `"Fix bug tomorrow #auth !high"`
→ `{ text: "Fix bug", dueDate: Date, tags: ["auth"], priority: "high" }`

## AI Integration

```tsx
<Btwfyi aiConfig={{ grokApiKey: process.env.GROK_KEY }} />
```

## Import/Export

- Paste JSON/Markdown into Command Palette
- Export via Management View (type "btwfyi")

## Command Palette

- `Alt + K` → Open
- Type `btwfyi` → Management view
- Paste list → Bulk import
