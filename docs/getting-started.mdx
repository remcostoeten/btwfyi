# Getting Started with Vigilo

Vigilo is a lightweight task awareness overlay that helps you track development tasks and connect them to DOM elements.

## Installation

### Package Manager

Choose your preferred package manager:

```bash
# pnpm (recommended)
pnpm add btwfyi

# bun
bun add btwfyi

# npm
npm install btwfyi
```

## Quick Setup

### React Integration

```tsx
import { Vigilo } from 'btwfyi/react'
import type { CategoryConfig } from 'btwfyi'

const categories: CategoryConfig[] = [
  {
    id: 'backend',
    displayName: 'Backend Tasks',
    items: [
      { text: 'Fix user authentication bug', action: 'fix', priority: 'high' },
      { text: 'Add API rate limiting', action: 'feat', priority: 'medium' },
    ],
  },
  {
    id: 'frontend',
    displayName: 'Frontend Tasks',
    items: [
      { text: 'Implement new dashboard', action: 'feat', priority: 'high' },
      { text: 'Add dark mode support', action: 'feat', priority: 'low' },
    ],
  },
]

function App() {
  return (
    <div>
      <Vigilo category="backend" instanceId="main-backend" categories={categories} />
      <Vigilo category="frontend" instanceId="main-frontend" categories={categories} />
    </div>
  )
}
```

### Vue Integration

```vue
<template>
  <div>
    <Vigilo category="backend" instance-key="main-backend" :categories="categories" />
    <Vigilo category="frontend" instance-key="main-frontend" :categories="categories" />
  </div>
</template>

<script setup>
import { Vigilo } from 'btwfyi/vue'
</script>
```

## Easy Styling

### Adjust Opacity and Colors via Props

```tsx
<Vigilo
  category="development"
  categories={categories}
  themeOverrides={{
    colors: {
      primary: '#10b981',           // Connector line color
      bgPanel: 'bg-zinc-900/80',    // Semi-transparent panel
    },
  }}
  stylesOverrides={{
    connectorDot: { r: 6 },         // Larger endpoint dots
  }}
/>
```

### Runtime Styling via Settings Panel

Users can adjust these settings by pressing **'s'** on the overlay:
- **Line Opacity** (0-100%)
- **Component Opacity** (10-100%)
- **Line Color** (color picker)
- **Toggle Lines** on/off
- **Toggle Badges** on/off

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `s` | Open settings menu |
| `?` | Show keyboard help |
| `/` | Focus search |
| `l` | Toggle connector lines |
| `b` | Toggle action badges |
| `Escape` | Close modals |
| `Alt+K` | Open command palette |
